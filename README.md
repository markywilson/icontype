# icontype

iconfont generator

## Installation

```sh
$ npm install icontype
```

## Usage

### in CLI

```sh
$ icontype --help
iconfont generator

Usage
  icontype [files] [options]

Options
  -h, --help       show help
  -v, --version    print version
```

#### Example:

```sh
icontype <files>
```

### in Node.js module

```js
const icontype = require('icontype');
```
